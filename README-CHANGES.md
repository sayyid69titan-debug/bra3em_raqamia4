# تعديلات الواجهة المطلوبة

تم تطبيق التعديلات التالية داخل المشروع:

1. إزالة عرض "ملخص الوحدات" النصي من صفحات الصفوف واستبداله بصور قابلة للاستبدال من مجلد:
   `public/assets`

2. تمت إضافة صور مستقلة لكل وحدة بصيغة SVG، ويمكن استبدالها مباشرة مع الحفاظ على نفس الاسم:
   `grade-1-unit-1.svg`
   `grade-1-unit-2.svg`
   ...
   وحتى صور وحدات الصف السادس.

3. حذف كلمة/تصور "العوالم" من محتوى الصفوف واستبداله بتسميات تعليمية مباشرة مثل:
   الأجزاء، الأنماط، التعليمات، الخوارزميات، البيانات، الأنظمة.

4. تحسين الألوان العامة للموقع لتصبح طفولية أكثر:
   خلفيات متدرجة، ألوان زاهية، بطاقات أكثر نعومة، أزرار ملونة، نقاط زخرفية.

5. إضافة GIF في رأس الصفحة الرئيسية:
   `public/assets/robot-solution.gif`
   ويظهر بجانبه النص: "الحل لمشاكلك هنا".

6. بقيت الألعاب والأنشطة وصفحات الصفوف موجودة، ولم يتم حذف وظائف التفاعل أو التقدم.
